## 📖 상세 기능 설명

### 🔐 로그인

- 닉네임 입력 시 **자동완성 드롭다운** (Firebase `allowedUsers` 목록 미리 로드)
- **6자리 입장코드** (password 타입) 입력 후 입장
- `localStorage` 기반 **자동 로그인** — 재접속 시 닉네임·코드 재입력 불필요
- 로그인 후 `roles` DB 조회 → 일반/연구원/관리자 역할 분기
- PC(≥900px) / 모바일(<900px) 레이아웃 자동 감지 및 적용
- **로그아웃**: 로컬스토리지 전체 초기화 후 새로고침

---

### 🏠 홈

- 닉네임 기반 환영 인사 (이름 + 👋)
- **내 시즌 전적 카드**: 승수·패수·전체 랭킹 순위 실시간 표시
- **최신 공지 카드**: 최신 공지 제목·날짜 표시, 탭 시 공지사항으로 이동, 읽지 않은 공지 🔔 NEW 배지 애니메이션
- **빠른 메뉴**: 길드전 공격, 추천 방어, PVE 공략 원터치 이동
- **홈 공성전 랭킹**: 요일(월~일) × 이번주/저번주/📈성장률 탭으로 전환하여 랭킹 미리보기
- **파괴신 시즌 총합**: 시즌별 길드 총점 추이 라인차트 + 최신 시즌 총점 표시
- 🌙/☀️ 테마 토글 버튼
- 관리자: ≡ 햄버거 관리자 메뉴 버튼

### ⚔️ 길드전 (`attackScreen`)

#### 방어팀 선택
- **퍼지 매칭(fuzzy match)** 검색 지원 드롭다운 (오타·공백 허용)
- 같은 이름의 방어팀이 여러 키에 분산 등록된 경우 **병합(merge)** 후 단일 항목으로 표시
- 방어팀 선택 시 해당 공격덱 목록 실시간 로드

#### 공격덱 목록
- **정렬**: 승률순 / 최신순 / 이름순
- 각 덱 버튼에 **승률 %** + **승패 수** 표시
- 덱 타입 배지: `⚡ 속공 따야 함` (주황) / `🛡️ 내줘도 됨` (파랑)
- 승률 70% 이상 시 🔥 이모지 자동 표시
- **내 덱편성 연동**: 같은 방어팀에 배정된 내 팀을 구분선 아래에 파란 카드로 표시

#### 공격덱 카드 (덱 선택 시 우측/하단 표시)
- 덱 이름, 제작자, **승률 배지** (승수·패수·승률%)
- 🔥 **중요 알림**: 관리자가 입력한 긴급 메모 (빨간 배너로 강조)
- 사용 타입 / 추천 반지 / 추천 펫 / 장비 세팅 / 스킬 순서
- 캐릭터 **진형 비주얼**: 전행·중행·후행 배치를 셀로 시각화 (관리자 직접 수정 가능)
- **최근 전적 도트**: 최근 10경기를 🟢W / 🔴L 색상 원으로 표시  
  → 마우스 호버 / 탭 시 닉네임·날짜·결과 툴팁 표시
- **승리 / 패배** 버튼으로 즉시 전적 기록
  - 패배 클릭 시 → **패배 기록 모달** 자동 열림
  - 성 선택 + 상대 닉네임 입력 후 → **⚡ 속공 계산 팝업** 연계 또는 **건너뛰기(기록만)**
- **🗡️ 덱편성에 추가**: 현재 선택한 방어팀·공격덱을 내 덱편성에 원클릭 추가
- 관리자 전용: `[관리자] 기록 초기화` 버튼 (승패 수 0으로 리셋)

#### 전적 기록 방식 3가지

| 방식 | 설명 |
|---|---|
| ① 공략 덱 선택 | 방어팀·공격덱 선택 후 승/패 버튼 클릭 |
| ② 직접 기록 | 공략에 없는 방어팀도 이름 직접 입력 후 기록 (`logs`에 `direct:true`로 저장) |
| ③ 없는 팀 기록 | DB에 없는 방어팀 상대 시 팀명 입력, 기록만 하거나 방어팀+공격덱 동시 제보 |

#### 제보 기능
- `+ 공격덱 제보`: 덱명·타입·반지·펫·진형·장비·스킬·승패 결과 포함하여 제보  
  → `suggestions/attack/{defenseKey}/{attackKey}` 저장, 관리자 승인 후 반영
- `📨 내 제보`: 내가 제보한 공격덱 목록 조회, 미승인 건 직접 **수정·삭제**
- 없는 팀 기록 시 **방어팀+공격덱 동시 제보** → `suggestions/defense_team` 저장

#### 관리자 패널 (연구원 이상)
- 방어팀 추가 (`defenseTeams` push) / 삭제 (7일 복구 로그 자동 저장)
- 공격덱 추가·수정·삭제 (삭제 시 7일 복구 가능)
- 공격덱 입력 항목: 이름·타입·중요알림·반지·펫·진형·장비·스킬
- 덱 이름에서 **캐릭터 자동 파싱** (`characters` 필드 자동 생성)
- 덱 데이터 **클립보드 복사/붙여넣기** (관리자 간 공유용)
- **캐릭터 역검색**: 특정 캐릭터 이름으로 해당 캐릭터가 포함된 공방덱 전체 검색

### 📅 오늘의 길드전 (`todayWarScreen`)

- **9개 성** 카드: 외성 1~5, 내성 1~3, 본성 — `todayGuildWar` DB 실시간 반영
- 성별 카드에서 **공격 기록**: 추천덱 팝업 → 공격덱 선택 → 승/패 → 상대 닉네임 입력
- **추천덱 팝업**: 해당 성 추천 공격덱 목록 + 현재 시즌 승률 높은 덱 TOP 5
- 기록 수정·삭제 가능
- **속공 기록 연계**: 패배 시 상대 속공 정보 기록 → 이후 공격 시 참고
- 전체 초기화 버튼 (관리자 전용)

#### 🔍 속공 상세보기 (신규)
- 각 상대 row에 **`🔍 분석` 버튼** 표시 (속공 데이터가 있는 경우)
- 클릭 시 바텀 시트 팝업으로 상세 정보 표시
  - 집계 속공 범위 (여러 제출 데이터 교차 계산 결과)
  - 제출 기록 건수 및 건별 상세: 기록자, 속공승/패, 추정 속공 범위, 기록 시각
  - **전투 순서 재현**: 아군(이름+속공) / 적군(이름+타입) 순서 그대로 시각화
- 길드원 누구나 열람 가능

### 🗡️ 길드전 덱편성 (`deckplanScreen`)

- **보유 반지 등록 패널** (토글 접기/펼치기): 보유 중인 반지 선택 후 저장
- **팀 카드 목록**: 팀마다 방어팀·공격팀 선택 + 반지 슬롯 3개 (각각 커스텀 라벨)
- 진형 선택: 기본·밸런스·공격형·보호형
- **드래그 앤 드롭** 팀 순서 변경 (마우스 드래그 + 가로 스크롤)
- 팀 추가 / 삭제
- 💾 저장 버튼: `deckPlan/{닉네임}` 에 저장
- **공유**: 제목 입력 후 스냅샷 저장 → `deckPlanShares` 에 저장  
  기존 공유 덮어쓰기 선택 가능
- 공유 목록: 타인 편성안 이름 클릭 → 티어별·팀별 상세 열람
- **승패 직접 기록**: 팀별 ✅/❌ 결과 반영, 팀당 승률 자동 계산
- **피드백 수신**: 관리자가 보낸 팀별 코멘트 확인
- **덱편성 알림 배지**: `deckPlanNotify` DB 기반, 새 피드백 수신 시 탭 배지

### 🛡️ 추천 방어 (`defenseScreen`)

- 관리자 등록 **추천 방어덱** 목록 (`recommendedDefenses` DB)
- 팀당 여러 **케이스(빌드)** 등록: 각 케이스마다 캐릭터 구성·진형·펫·스킬 독립 저장
- 케이스 탭 전환으로 세부 공략 확인
- **진형 비주얼** 렌더링 (캐릭터명 기반 전·중·후 자동 배치)
- 검색 필터 지원
- `📨 내 방어덱 제보`: 방어덱 정보 제보 (`suggestions/defense` 저장)  
  내 제보 목록 조회·수정·삭제
- 관리자 패널: 방어팀 추가·수정·삭제, 케이스 추가·수정·삭제

### ⚡ 속공 계산기 (전체화면 `speedCalcScreen` + 팝업 모달)

두 버전 모두 동일한 로직, 팝업은 패배 기록과 자동 연계

- **캐릭터 슬롯**: 아군 최대 3개 · 적군 최대 3개
- 슬롯별 팀(아군/적군) · 캐릭터명 · 속공 수치 직접 입력
  - **아군**: 이름 + 속공 수치 직접 입력
  - **적군**: 이름 + 타입 선택 (속공은 전투 순서 기반으로 자동 추론)
- **빠른 선택 버튼**: 현재 선택된 공격덱·방어덱 캐릭터를 원터치로 슬롯에 추가
  - **토글 방식** — 한 번 클릭하면 슬롯에 추가(버튼 진하게 + ✓ 표시), 다시 클릭하면 슬롯에서 제거
  - 전체 초기화 후에도 버튼이 즉시 활성화 상태로 복구됨
- **기타 캐릭터 검색**: 이름 입력 → DB 자동완성 → 클릭으로 슬롯 추가
- 전체 슬롯 초기화 버튼 — 초기화 시 빠른 선택 버튼 상태도 함께 리셋
- **🔍 분석하기**: 아군·적군 합산 속공 비교 → 선공 여부 판정, 적군 속공 범위 자동 추론
- 분석 완료 시 **오늘의 길드전 등록 미리보기** 섹션 자동 표시
  - 등록 내용 미리보기: 성, 상대 닉네임, 총속공 범위, 개별 캐릭터 범위를 텍스트 + 칩으로 표시
- **속공 승패** 선택 (승리/패배)
- **💾 저장하기** (팝업 버전): 분석 결과 + 패배 기록(성, 상대 닉네임, 사용 공격덱) 연계 저장  
  → 저장과 동시에 **오늘의 길드전에 자동 등록** (길드원 전체 공개)
- 전체화면 버전: 결과 저장 후 `logs` DB 에 별도 기록

### 🧮 딜 계산기 (`calcScreen`)

세트 프리셋별·보스별 정밀 딜량 시뮬레이터

**입력 항목**

| 범주 | 세부 항목 |
|---|---|
| 캐릭터 | 공격력, 크리 확률, 크리 배율, 약점 확률, 약점 배율, 속성 보정 |
| 펫 | 펫 종류별 공격력 보정 선택 |
| 버프 | 인카운터 버프, 길드 버프, 기타 버프 수치 |
| 스킬 | 스킬 배율 직접 입력 |
| 상대 정보 | 방어력, 방어 감소율, 추가 보정 |

- 세트 **프리셋 선택** (미리 저장된 세트 불러오기)
- **보스 모드 토글**: 보스 전용 추가 배율 적용
- 크리/약점 확률 **슬라이더** 조절
- 결과: 평균 딜 · 최대 딜 · 크리 발생 시 딜 **실시간 계산**
- ❓ 가이드 팝업 (계산 방식 설명)

### 🏆 랭킹 (`rankingScreen`)

#### 길드전 랭킹
- `logs` DB 기반 시즌 누적 **승·패·승률** 개인 순위
- 1~3위 금·은·동 특별 표시
- 닉네임 클릭 → 날짜별 상세 전적 펼침 (날짜·방어팀·공격덱·결과)
- **일별 필터** (전체 + 요일 탭)

#### 공성전 랭킹
`pveScores/siege/{요일}/{이번주|저번주}` 구조

- 이번 주 / 저번 주 탭 전환
- **이번 주 → 저번 주 주차 이동** 버튼 (관리자)
- 점수 입력 방법 3가지:
  1. **닉네임 자동완성** + 점수 직접 입력
  2. **대량 텍스트** 붙여넣기 (줄바꿈 구분, 닉네임 자동완성 지원)
  3. **🤖 AI 스캔**: 스크린샷 이미지 업로드 → Claude API가 점수 자동 인식·입력
- 길드 **총합 점수** 표시
- 점수 개별 삭제 가능

#### 파괴신 랭킹
`pveScores/advent/{보스}/{이번주|저번주|seasons}` 구조

- 보스별 탭 전환
- AI 스캔 지원 (공성전과 동일)
- 주차 이동 (관리자)
- **시즌 합산 추가**: 현재 주를 시즌으로 집계하여 누적
- **시즌 총합 라인차트**: 시즌별 길드 총점 추이 시각화 (홈 화면과 연동)
- 점수 개별 삭제 가능

### ⚔️ 총력전 (`totalwarScreen`)

`totalwar/teams/{닉네임}/{tierId}` 구조로 닉네임별 독립 저장

- **티어 선택 탭**: 등록된 티어 목록 가로 스크롤, 티어별 독립 팀 구성
- **팀 카드 가로 스크롤** 그리드 (드래그 스크롤 + 마우스 휠)

#### 팀 편성 모달 (슬롯 클릭 or ✏️ 수정 버튼)
- **진형 선택**: 기본(후3전2) · 밸런스(후2전3) · 공격(후4전1) · 보호(후1전4)
- **캐릭터 슬롯 5개**: 클릭 → 선택 패널 열림, DB 검색·자동완성
  - 채워진 슬롯 **꾹누름(0.6초)** → 스탯·장비 입력 모달
  - 채워진 슬롯 **더블탭/더블클릭** → 편성에서 제거
- **속공** 수치 입력
- **펫** 검색 자동완성 선택
- **스킬 순서** 텍스트 입력
- **팁/메모** 입력
- 저장 시 Firebase에 즉시 반영

#### 캐릭터 스탯 입력 모달 (꾹누름으로 진입)

| 탭 | 입력 항목 |
|---|---|
| 📊 스탯 | 물리공격력·방어력·생명력·속공·약점확률·막기확률·받는피해감소·효과적중·치명타확률·효과저항·치명타피해·별 수(클릭 토글)·초월(오버클락) 단계 |
| ⚔️ 무기 | 무기1·무기2 각각 세트 + 메인옵션 선택 |
| 🛡️ 방어구 | 방어구1·방어구2 각각 세트 + 메인옵션 선택 |
| 💍 반지 | 반지 선택·성급(4/5/6성)·강화 수치·세공 옵션 텍스트 입력 |

- 공유된 카드에서도 **스탯 정보 읽기전용 뷰** 팝업

#### 팀 카드 관리
- **드래그 앤 드롭** (터치/마우스) 팀 순서 변경
- 팀 **복사/붙여넣기** (같은 티어 내)
- 팀 추가 / 삭제

#### 공유
- 🔗 공유 버튼 → 제목 입력 → `totalwar/shares` 저장
- 기존 공유 **덮어쓰기** 선택 가능
- 하단 **공유 목록** 인라인 표시: 이름 클릭 → 티어 탭 전환하며 상세 확인
- 공유 카드에서 캐릭터 꾹누름 → 스탯 정보 읽기전용 팝업
- 공유 삭제 (본인 것만)

### 🏰 PVE 공략 (`pveScreen`)

`pveData/{던전타입}/{던전명}/{빌드키}` 구조

- 던전 탭별 공략 카드 목록
- **빌드 선택**: 던전당 여러 빌드 등록 (예: 기본빌드, 카일있음, 속공용)  
  빌드 탭 클릭으로 전환, 세부사항(태그) 선택으로 세부 공략 표시
- **진형 비주얼**: 하나의 빌드에 진형 여러 개 등록 가능, 각각 이름과 캐릭터 배치 표시
- 덱 조합 / 스킬 순서 / 장비 세팅 텍스트 표시
- **내 공성 점수 등록** (랭킹 화면과 연동):
  - 현재 등록된 이번 주 점수 표시
  - 점수 입력 후 등록 → `pveScores/siege/{요일}/{week}` 에 자동 저장·갱신
- ❓ 페이지 가이드 팝업
- 관리자 수정 모드:
  - 빌드 이름 변경, 새 빌드 추가, 현재 빌드 삭제
  - 세부사항(태그) 추가·삭제
  - 진형 추가·수정·삭제 (여러 진형 순서 관리)
  - 덱 조합·스킬 순서·장비 세팅 텍스트 편집
  - 데이터 **복사/붙여넣기** (다른 던전에 동일 빌드 적용)
  - 빌드 순서 정렬 저장

### 📢 공지사항 (`noticeScreen`)

`notices/{key}` 구조

- **공지 상세 팝업**: 본문 + 이미지 그리드(최대 3장), 이미지 탭 시 **전체화면 확대**

- **자동 팝업**: 관리자가 특정 공지를 팝업으로 설정 시 앱 진입 시 자동 표시
- 관리자: 공지 작성 (제목·본문·이미지 최대 3장 업로드), 숨기기/보이기 토글, 삭제
- 관리자: **읽음 현황 조회** — 각 공지를 읽은 길드원 닉네임 목록

### 👥 길드원 관리 (`adminMembersScreen`)

- 닉네임 **역할 부여**: 일반 / 연구원 / 관리자 → `roles/{nick}` 저장
- 닉네임 **변경**: 기존 닉네임 자동완성, `allowedUsers` + `roles` + 관련 데이터 일괄 변경
- **입장코드 발급**: 6자리 코드 생성 → `allowedUsers/{nick}` 저장
- 전체 길드원 목록 조회·검색

---

### 📚 캐릭터/펫 DB (`adminDatabaseScreen`)

- **신규 캐릭터 등록**: 이름·속공 수치·타입 → `characters/{name}` 저장
- 캐릭터 목록: 타입 수정, 삭제
- **방어덱 캐릭터 분석**:
  - 등록된 모든 방어팀·공격덱 이름을 토큰화하여 `characters` DB와 매칭
  - 완전 매칭 / 부분 매칭 / 매칭 실패 항목 분류 표시
  - 미등록 캐릭터 감지 → 일괄 DB 등록 (`fillMissingCharacters`)
  - 매칭된 캐릭터를 덱 데이터 `characters` 필드에 자동 적용 (`applyCharactersToDecks`)
- **신규 펫 등록**: 이름 → `pets/{name}` 저장, 펫 목록 관리·삭제

---

### 📋 제보 관리 (`adminReportsScreen`)

- **공격덱 제보** (`suggestions/attack`): 상세 편집 모달 → 승인(공식 DB 반영) / 거절(전적 로그만 남김)
- **방어덱 제보** (`suggestions/defense`): 승인 → `recommendedDefenses` 에 반영 / 거절
- **상대 방어팀덱 제보** (`suggestions/defense_team`): 없는 팀 제보 관리
- **미확인 제보 배지**: `badge-reports` — 새 제보 수신 시 관리자 탭에 숫자 표시

### 📊 승패 관리 (`adminScoresScreen`)

- 닉네임 검색 + 자동완성 → 개인 승·패 **+1/-1 수동 조정**
- 개인 전적 **전체 삭제** (해당 닉네임의 `logs` 항목 일괄 삭제)
- **시즌 초기화**: `logs` 전체 삭제 (확인 다이얼로그 필수)

### ♻️ 데이터 복구 (`adminRecoveryScreen`)

- 방어팀·공격덱 삭제 시 **자동으로 `deletionLogs` 에 백업** (7일 보관)
- 7일 이상 된 로그는 `cleanupOldDeletionLogs()` 로 자동 삭제
- 복구 목록에서 항목 클릭 → **원클릭 복구** (원래 키로 데이터 재저장)
- **수동 복구 패널**: 고급 사용자용 직접 복구 UI

`todayDeck/decks` + `todayDeck/enabled` 구조

- **팝업 자동 표시 ON/OFF**: 앱 진입 또는 오늘의 길드전 진입 시 추천덱 팝업 자동 표시 여부
- 등록된 추천덱 목록 확인 및 삭제

## 🚀 배포 방법

1. Firebase 콘솔에서 프로젝트 생성 후 **Realtime Database** 활성화
2. HTML 파일 내 `firebaseConfig` 객체를 본인 프로젝트 값으로 교체
3. (선택) Anthropic API 키를 AI 스캔 기능 부분에 설정
4. `index_updated.html` 파일을 정적 호스팅 서비스에 업로드  
   (Firebase Hosting · GitHub Pages · Cloudflare Pages 등 모두 가능)
5. 관리자 계정으로 첫 로그인 → 길드원 닉네임·입장코드 발급 → 공유

**UX 개선**

- 빠른 선택 버튼을 **토글 방식**으로 변경
  - 한 번 클릭 → 슬롯에 추가, 버튼 진하게 강조 + ✓ 표시
  - 다시 클릭 → 해당 캐릭터 슬롯에서 제거, 버튼 원래 상태로 복구
  - 기존: 슬롯에 있으면 `disabled` + 취소선으로 고정, 제거 불가
  - 변경: 전체 초기화 없이 개별 캐릭터 단위로 추가/제거 가능
