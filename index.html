⚔️ 캐릭터
프로필 공격력
캐릭터 클릭 → 스탯창에 표시되는 공격력 숫자 그대로 입력
태생 공격력 (도감)
도감에서 해당 캐릭터 클릭 → 30레벨 5성강 기준 공격력 수치.
장비·버프 없는 순수 기본 공격력
진형 보너스 %
현재 진형이 주는 공격력 증가 %. 예) 기본진형 후방이면 공격 진형 보너스 확인 후 입력. 없으면 0
🐾 펫
펫 깡공
펫 정보창에서 확인. 펫이 주는 고정 공격력 수치. 펫 없으면 0
펫 잠재력 %
펫 잠재력으로 올린 공격력 % 합산. 태생 공격력에 곱해짐. 펫 없으면 0
✨ 버프
공격력 버프 %
전투 중 받는 공격력 증가 버프 전체 합산.
예) 물리피해량 증가 스킬, 아군 버프 스킬 등 모두 더해서 입력
💥 스킬
스킬 계수 %
스킬 정보창에 표시된 배율 숫자. 예) "공격력의 320%" → 320 입력. 기본 공격이면 100
치명타 피해 %
스탯창 치명타 피해 수치 + 치명타 피해 버프 합산. 치명타가 터질 때 적용. 기본값은 150
약공 계수
약점 공격 시 배율. 기본 1.3 / 추적자 4세트 착용 시 1.5 / 약점 없으면 1.0
약점공격 피해증가 %
약점공격 피해 증가 버프 합산. 예) 약공 피해 23% 버프 → 23 입력. 없으면 0
공격 횟수
멀티히트 스킬의 총 타수. 단타 스킬이면 1. 예) 3타 스킬이면 3 입력
👾 상대 정보
방어력
보스·적의 방어력 수치. 공략 정보나 커뮤니티에서 확인. 모르면 0
방증 %
적이 받는 방어력 증가 효과. 진형 보너스로 적 방어 오를 때 입력. 없으면 0
방감 %
방어력 감소 버프 합산. 아군 스킬 중 적 방어 낮추는 효과 모두 더해서 입력. 없으면 0
방무 %
방어 무시 합산. 장비·스킬·버프로 얻는 방어무시 % 모두 더해서 입력. 없으면 0
취약 %
물리·마법 취약 버프 합산. 적에게 취약 디버프 걸었을 때 수치 입력. 없으면 0
피해량 증가 %
보스 추가 배율, 반지 피해 증가, 기타 피해량 증가 버프 모두 합산. 없으면 0
💡 계산 팁
• % 입력란에는 숫자만 입력 (예: 23% → 23)
• 모르는 값은 0으로 두면 제외됨
• 치명타 계산은 치명타가 터진다는 가정하에 계산
• 결과는 예상 데미지이며 게임 내 편차 있을 수 있음
